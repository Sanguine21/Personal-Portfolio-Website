# Personal Portfolio Website

A modern and responsive **personal portfolio website** designed to showcase my profile, skills, and projects as an aspiring **Web Designer**.

This portfolio highlights my work, learning journey, and technical abilities in a clean, structured, and recruiter-friendly format.

The goal of this project is to create a professional online presence where I can present my projects, demonstrate my design skills, and share my contact information for opportunities and collaborations.

---

## Features

- Responsive design that works on desktop, tablet, and mobile devices  
- Modern UI layout with clean typography and structured sections  
- Personal introduction and profile section  
- Skills and expertise display  
- Project showcase with preview images  
- Live demo and repository links for projects  
- Clickable contact options (email, phone, LinkedIn, GitHub)  
- Downloadable resume option  
- Smooth navigation across sections

---

## Sections Included in the Website

### Home
The hero section introduces my profile as an aspiring web designer and provides quick navigation to projects and resume download.

### About
A short description about me, my interests in web design, front-end development, content writing, and my learning journey.

### Skills
Displays the technologies and tools I am currently learning and using.

Example skills:

- HTML  
- CSS  
- JavaScript  
- Responsive Web Design  
- UI Design  
- Figma  

### Projects
A section showcasing selected projects with preview images and descriptions.

Projects included:

- **Personal Portfolio Website**  
- **Digital Scam Detection Project**

Each project contains links to:

- Live Demo  
- GitHub Repository / Case Study  

### Journey
Shows my learning path and academic progress as I continue developing my skills in web design and development.

### Contact
Provides ways to connect with me through:

- Email  
- Phone  
- LinkedIn  
- GitHub  

A **Download CV** button is also included.
